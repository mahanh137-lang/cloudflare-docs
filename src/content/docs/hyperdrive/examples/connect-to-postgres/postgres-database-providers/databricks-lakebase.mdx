This example shows you how to connect Hyperdrive to a [Databricks LakeBase](https://www.databricks.com/product/lakebase) database using PostgreSQL native role authentication.

## 1. Configure LakeBase for PostgreSQL authentication

Databricks LakeBase supports PostgreSQL-compatible connections using native role authentication, eliminating the need for service principals.

### Enable PostgreSQL native role login

1. In your Databricks workspace, navigate to **SQL Warehouses** or **Compute**.
2. Select your SQL warehouse or compute cluster that will host your LakeBase database.
3. Go to **Configuration** > **Advanced options**.
4. Enable **PostgreSQL native role login** in the configuration settings.
5. Save the configuration and restart the compute resource if required.

### Create your user

Once your database is created, you will need to create a user for Hyperdrive to connect as. Although you can use the **Master username** configured during initial database creation, best practice is to create a less privileged user.

To create a new user, log in to the database and use the `CREATE ROLE` command:

```sh
# Log in to the database
psql postgresql://MASTER_USERNAME:MASTER_PASSWORD@ENDPOINT_NAME:PORT/database_name
```

Run the following SQL statements:

```sql
-- Create a role for Hyperdrive
CREATE ROLE hyperdrive;

-- Allow Hyperdrive to connect
GRANT CONNECT ON DATABASE postgres TO hyperdrive;

-- Grant database privileges to the hyperdrive role
GRANT ALL PRIVILEGES ON DATABASE postgres to hyperdrive;

-- Create a specific user for Hyperdrive to log in as
CREATE ROLE hyperdrive_user LOGIN PASSWORD 'sufficientlyRandomPassword';

-- Grant this new user the hyperdrive role privileges
GRANT hyperdrive to hyperdrive_user;
```

## 3. Connect using PostgreSQL drivers

When connecting to LakeBase through Hyperdrive, use standard PostgreSQL drivers:

- [node-postgres (pg)](/hyperdrive/examples/connect-to-postgres/postgres-drivers-and-libraries/node-postgres/)
- [Postgres.js](/hyperdrive/examples/connect-to-postgres/postgres-drivers-and-libraries/postgres-js/)
- [Drizzle ORM](/hyperdrive/examples/connect-to-postgres/postgres-drivers-and-libraries/drizzle-orm/)

These drivers will work seamlessly with LakeBase's PostgreSQL-compatible interface through Hyperdrive's connection pooling and query optimization.
